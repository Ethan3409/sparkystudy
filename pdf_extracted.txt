Class 1 and Class 2 Circuits

Rationale
Why is it important, (for you to learn this skill?

Some circuits do not have the same wiring requirements as li ighting and power branch
circuits. The circuits described in Section 16 of the Canadian Electrical Code (CE Code)
can be supplied by conductors smaller than No. 14 AWG or with less restrictive wiring
methods. This is permitted because the circuit current and/or voltage values are limited to
levels that do not normally present serious shock or fire hazards,

Although No. 14 AWG is the minimum conductor size for most types of circuits, you
may be permitted to install conductors smaller than No. 14 AWG for certain circuits.
These circuits include a thermostat, doorbell, or security system when wiring a house;
heating, ventilation, air conditioning, and refrigeration (HVAC/R); or motor control
wiring in a commercial or industrial setting.

Outcome

When you have completed this module, you will be able to

Determine CE Code requirements for Class 1 and Class 2 circuits.

Objectives

Introduction

The circuits discussed in this module are described in Section 16 Class 1 and Class 2
circuits. The classifications of these circuits are determined by their source limitations.
This module provides a description of these circuits and their installation requirements.
