📋 CEC Code Limits

The National Electrical Code (NEC) specifies that conduit fill cannot exceed 40% for 3 or more conductors.

Each conductor type has a specific cross-sectional area (mm²) that counts toward the total fill.

Overfilling reduces cooling airflow and damages conductors!
