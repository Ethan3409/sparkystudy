Transformer Equations

Voltage Relationship: V1 / V2 = N1 / N2
Current Relationship: I1 / I2 = N2 / N1 (Inverse)
Power Relationship: P1 = P2 (Ideal transformer, no losses)
Impedance Reflection: Z_reflected = (N1/N2)² × Z_load
